# SukiCart API

A modern E-Commerce API built for scalable online shopping applications.
SukiCart API provides secure authentication, product management, cart handling, order processing, and Cash on Delivery (COD) checkout functionality for web and mobile commerce platforms.

SukiCart Client App

A modern E-Commerce Client Application for browsing products, managing carts, and placing Cash on Delivery (COD) orders seamlessly.

SukiCart Client App connects directly to the SukiCart API to provide a fast, responsive, and user-friendly shopping experience for customers.
